# What is Copilot? 
Co-Pilot is an AI-powered coding agent available as extension within the IDE. 

Example : Github Copilot

# How Github Copilot will work? 
GitHub Co-Pilot is powered by large language models (LLM's), and it is going to understand your current file, your project structure, your existing framework pattern, and the comments written by the user and the method names and all the libraries imported within the project. 

# Different features of Copilot ?
1. Inline code  suggestions : Whenever we are going to write the code, it is going to fill most of the part by itself. 

2. Co-pilot chat : Copilot Chat is going to give you a chat window that is going to help you to explain the code and refactor the code to optimize the existing solutions. It is also going to help you to debug the program. 

3. Agent Mode : Agent mode is all about. Instead of simply generating the code, it is going to create the file, update the code, fix the project by itself, and it is also going to execute the commands and generate the frameworks. 

4. Test Generation : It can actually generate all the possible test cases, assertions, and reusable methods during the automation development process. 

5. Generating the documentation : So, Copilot can help us to add the detailed documentation comments for each and every common method that we are creating. 

6. MCP Integration : Copilot has the capability to integrate MCP to complete the task and to handle the tools as well.
